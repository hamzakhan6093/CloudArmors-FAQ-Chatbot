# CloudArmors-FAQ-Chatbot
An AI-powered FAQ Chatbot for Cloud Armors that answers queries about services, departments, hiring, support hours, office details, cloud and DevOps solutions, AI offerings, and more. This repository includes an organized dataset, training files, and resources for building an intelligent company assistant.
